**ip-to-geo**
---
**Возможности**

- Получение информации о:
  - Стране
  - Регионе
  - Городе
  - Провайдере
  - Организации
  - Часовом поясе
  - Географических координатах (широта и долгота)
- Проверка введённого IP-адреса
- Обработка ошибок и неправильного ввода
- Открытие результата в браузере

---
**Установка и запуск**

1. Клонируй репозиторий
2. Запусти файл
3. Введи ip
